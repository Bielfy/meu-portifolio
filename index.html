Projetos

Consulta e Cadastro de Endereços via CEP (Java + ViaCEP)

Aplicação em Java que integra com a API ViaCEP para buscar endereços a partir de CEPs digitados no terminal e salvar os dados em um arquivo .json. Utiliza Maven, Gson e tratamento de exceções para garantir robustez e organização no processo.

Consulta de Filmes via API OMDB (Java + JSON)

Aplicação em Java que permite buscar filmes usando a API OMDB diretamente pelo terminal. As informações coletadas (nome, ano e duração) são armazenadas em um arquivo .json. Projeto com foco em integração de APIs, uso do Maven e tratamento de erros.

Tela de Login Simples

Uma interface de login funcional com HTML, CSS e JavaScript, incluindo validação de entrada e autenticação básica com credenciais fixas. Projeto focado em usabilidade e estruturação front-end clara.
